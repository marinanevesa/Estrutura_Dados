# Estrutura de Dados I - Uni-FACEF

Repositório de implementações práticas desenvolvidas durante as aulas na **Uni-FACEF**, com foco no estudo de algoritmos e estruturas de dados dinâmicas.

## 🚀 Conteúdo
* **Classes:** Fundamentos de POO em JS.
* **Pilhas (Stack):** Conceitos de LIFO.
* **Filas (Queue) e Deques:** Conceitos de FIFO.
* **Listas Encadeadas:** Estruturas dinâmicas.

## 🛠️ Tecnologias
* **Linguagem:** JavaScript (MJS)
* **Ambiente:** Node.js
